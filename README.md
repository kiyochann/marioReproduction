# Super Mario Bros. 1-1 Reproduction

このプロジェクトは、初代スーパーマリオブラザーズの 1-1 ステージをブラウザ上で忠実に再現した、プレイ可能なデモプログラムです。
特にゴールシーケンス（ポールへの飛びつきから旗の降下、自動歩行まで）の挙動にこだわって調整されています。

## 実行方法

1.  このフォルダ全体を保存します。
2.  Windows の場合は、`PLAY_GAME.bat` をダブルクリックしてください。
3.  または、ブラウザで直接 `index.html` を開いてください。

## 操作方法

*   **移動**: 左右方向キー（`←` `→`）
*   **ジャンプ**: スペースキー または `↑` キー
*   **ダッシュ / ファイアボール**: `Shift` キー または `Z` `X` キー
*   **しゃがむ**: `↓` キー（スーパーマリオ時）
*   **土管に入る**: 土管の上で `↓` キー（特定の土管のみ）

## 主な機能と調整

*   **忠実なゴールシーケンス**: 旗の降下タイミング、マリオのポール滑り、土台での停止、自動歩行への切り替えを細部まで再現。
*   **物理演算**: ジャンプの高さ調節や慣性など、オリジナルの操作感を追求。
*   **資産管理**: `assets` フォルダ内の画像素材を動的に読み込み、16x16 タイルベースで描画。
